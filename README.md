# 📚 Repositório de Atividades — 1º Semestre  
## Curso Técnico em Desenvolvimento de Sistemas — SENAI Tambaú

Este repositório reúne todas as atividades desenvolvidas durante o **primeiro semestre** do curso técnico em Desenvolvimento de Sistemas no SENAI (unidade Tambaú).

---

## 🎯 Objetivo

Organizar e documentar os exercícios, projetos e práticas realizadas ao longo do semestre, facilitando o acompanhamento da evolução no aprendizado das seguintes áreas:

- Lógica de Programação  
- Algoritmos  
- Desenvolvimento Web  
- Banco de Dados  
- Versionamento com Git  

---

## 🧠 Conteúdos Abordados

### 🔹 Lógica de Programação
- Estruturas condicionais (`if`, `else`, `switch`)
- Estruturas de repetição (`for`, `while`)
- Variáveis e tipos de dados
- Fluxogramas e pseudocódigo

### 🔹 Algoritmos
- Resolução de problemas computacionais
- Organização de código
- Boas práticas

### 🔹 Desenvolvimento Web
- HTML5
- CSS3
- Introdução ao JavaScript

### 🔹 Banco de Dados
- Modelagem de dados
- SQL básico (`SELECT`, `INSERT`, `UPDATE`, `DELETE`)
- Criação de tabelas

### 🔹 Versionamento
- Git e GitHub
- Commits
- Branches
- Pull Requests
